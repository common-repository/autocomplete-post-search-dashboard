== Description ==
This plugin puts an autocomplete search box on a post/page/custom post type editor in the admin area and allows user to jump to any post or page.

This plugin puts an autocomplete search box on a post/page/custom post type editor in the admin area.

It allows you to jump between posts without having to go back to the post list and search from there.

== Screenshots ==

1. Search Box in Post Screen Area

<strong>Feature</strong>

* This plugin creates a search box in post screen area in upper right side.
* As you type any post name, it will suggest you related post with ids by autocomplete functionality.
* It supports pages, posts and custom post type.

== Installation ==
1. Upload the folder "jump-to-post-page" to "/wp-content/plugins/" path.
2. Activate the plugin through the "Plugins" menu in WordPress backend
3. After activating the plugin, a search box will appear in post/page screen area in right hand side in admin.
4. Enjoy!
